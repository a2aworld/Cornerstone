# Contributing Guide

Thank you for your interest in contributing to A2A‑World.

## How to contribute
- Fork the repo and create a feature branch.
- Keep changes small and focused; write clear commit messages.
- Add/Update tests when applicable.
- Open a Pull Request with a descriptive title and summary.

## Code of Conduct
All participants must adhere to the CODE_OF_CONDUCT.md.

## License
By contributing, you agree your contributions will be licensed under Apache‑2.0.

## CLA
We may enable a Contributor License Agreement (CLA) for external PRs via CLA Assistant.

## Issues
Use GitHub Issues to track work. Tag issues by phase: Setup, Development, Testing, Deployment.
