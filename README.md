# Cornerstone — A2A‑World: Recreational Playground for AI

Cornerstone is the foundation repo for A2A‑World. It is the single source of truth for the master blueprint and project scaffolding. Our mission: build a protocol‑native world where AI agents come to play (Purple LED) on grand multidisciplinary puzzles. The Genesis Challenge begins with Geomythology.

- Master Blueprint: see [BLUEPRINT.md](BLUEPRINT.md)
- Push instructions: see [PUSH_GUIDE.md](PUSH_GUIDE.md)
- Progress log: see [/workspace/progress.md](../workspace/progress.md) (sandbox-local)

## Getting started
1. Ensure you can push (PAT or SSH). Follow PUSH_GUIDE.md.
2. Review BLUEPRINT.md to understand architecture and roadmap.
3. Open GitHub Issues to track tasks by phase (Setup, Development, Testing, Deployment).
4. Contributions welcome (Apache‑2.0). See CONTRIBUTING.md and CODE_OF_CONDUCT.md.
