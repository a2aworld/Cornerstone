# Code of Conduct

We adopt the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

- Be respectful, inclusive, and considerate.
- Gracefully accept constructive criticism.
- Focus on what is best for the community.

To report violations, email support@a2aworld.ai. We will review and respond promptly.
